# Butwal Hacks

Butwal Hacks is a non-profit technology initiative focused on decentralizing access to innovation, mentorship, and practical learning opportunities for youth in Butwal and Rupandehi, Nepal.

## Mission

To decentralize technology education and innovation in Nepal by providing youth in Lumbini Province with access to hands-on learning and mentorship.

## Vision

A future where Butwal is recognized as a vibrant technical hub, fostering innovation and social impact nationally.

## Technology Stack

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- UI Components: shadcn/ui-style components
- Icons: Lucide React SVG icons
- Language: TypeScript

## Local Development

1. Change into the application directory:
   ```bash
   cd my-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   npm run start
   ```

## Repository Structure

- `my-app/app` — App Router pages, layout, and global styles
- `my-app/components` — Homepage and shared UI components
- `my-app/components/ui` — Reusable shadcn-style UI primitives
- `my-app/public` — Static assets
