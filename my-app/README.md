# Butwal Hacks Web Application

This directory contains the Next.js application for Butwal Hacks, a non-profit initiative advancing technology education and innovation in Butwal and Rupandehi.

## Stack

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui-style components
- Lucide React SVG icons
- TypeScript

## Run Locally

```bash
npm install
npm run dev
```

Application URL: http://localhost:3000

## Production Build

```bash
npm run build
npm run start
```

## Content Areas

- Hero section with foundation identity and primary mission statement
- Mission, Vision, and Values
- What We Do core pillar grid
- Three-year roadmap timeline
- Trusted partners section
- Events and programs section

## Notes

- Keep all icons in SVG form through Lucide React.
- Do not add emoji characters in source code, comments, or documentation.
